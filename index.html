Slack Christmas Gift

A bot that uses Slack API to plan a random one-to-one gift for your team's christmas party.

To use it just add the integration to your slack's team directory by clicking on the "Add to slack" button.

Once the integration is done just use the following command in any channel : 

/christmas

Santa Bot will send a message to each of the channel members to personnally inform them who they haft to gift for the party.
